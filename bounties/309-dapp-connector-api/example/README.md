# Midnight DApp Connector API Example

This example demonstrates how to build a browser dApp that connects to Midnight wallets using the DApp Connector API.

## 📦 Installation

```bash
npm install
```

## 🚀 Development

```bash
npm run dev
```

## 🔧 Build

```bash
npm run build
```
